## Color Palette Used

- **Navy (#071A2B):** Hero background, headings, company overview section
- **Teal (#00D6B3):** Logo, bubble borders, highlights, primary buttons
- **White (#FFFFFF):** Text on dark backgrounds, section backgrounds
- **Light Gray (#F1F1F1):** Alternating section backgrounds
- **Charcoal (#2C3E50):** Body text

## Typography Hierarchy

- **H1 (48px, Montserrat Bold):** Hero headings, case study titles
- **H2 (32px, Montserrat SemiBold):** Section headings
- **Philosophy labels (28px, IBM Plex Serif Italic):** Philosophy quotes
- **Body (16-18px, IBM Plex Sans Regular):** All body copy
- **Buttons (16-18px, Montserrat SemiBold):** CTAs

## Interactive Elements

1. **Bubbles:** Hover effect scales up 5%, border turns white, "See solution" button appears
2. **Primary button:** Hover inverts colors (transparent bg, teal border)
3. **Secondary button:** Hover fills with navy background
4. **Smooth scroll:** For internal anchor links

## Responsive Breakpoints

- **Desktop:** 1200px+ (default layout)
- **Tablet:** 768px-1199px (adjusted spacing)
- **Mobile:** < 768px (stacked bubbles, smaller text, single-column testimonials)
